# Airport TSA Times — Site Map

## Tech Stack
- React 19 + TypeScript + Vite 8
- Tailwind CSS v4 (CSS-based config via `@theme` in index.css)
- React Router v7 (client-side routing)
- Supabase (PostgreSQL, RLS policies, real-time subscriptions)
- Google Fonts: Inter + JetBrains Mono

## Routes
- `/` — Home dashboard (airport grid + live feed)
- `/airport/:code` — Airport detail (e.g. `/airport/jfk`)
- `/report` — Report wait time (optional `?airport=LAX` preselect)

## Design System
- **Theme**: Warm editorial — cream backgrounds, coral accents, monospace numbers
- **Colors**: cream (#FAF9F6), surface (#FFFFFF), ink (#1a1715), coral (#E8553A)
- **Wait colors**: green (<15m), amber (15-30m), red (>30m)
- **Typography**: JetBrains Mono for numbers/codes (`.mono`), Inter for body
- **Cards**: White surface with soft shadows, border-light borders, rounded-2xl
- **Buttons**: Pill-shaped (rounded-full), coral primary, segmented sort controls

## Data Flow
- Supabase real-time subscriptions (INSERT events on wait_reports)
- Views: `current_wait_times` (4hr window), `checkpoint_wait_times`
- RLS: public read all, public insert wait_reports (rate-limited)
- Rate limit: max 5 reports per IP per hour (PostgreSQL trigger)
- Twitter ingestion: `scripts/ingest-tweets.ts` (SocialData API, NLP extraction)

## Features
- **Geolocation**: "Near me" button on home page, finds nearest airport via Haversine distance
- **Checkpoint filtering**: Click checkpoint on detail page → filters live feed to that terminal
- **Animated planes**: SVG animateMotion with contrails, above background but behind cards (z-[1])
- **Bottom tab bar**: 2-tab nav (Airports/Report) with large touch targets (flex-1, w-14 h-10 pills)

## Gotchas
- Tailwind v4 uses CSS-based config (`@theme` in index.css), NOT tailwind.config.js
- PostCSS plugin is `@tailwindcss/postcss`, not `tailwindcss`
- Supabase anon INSERT with `return=representation` fails — use `return=minimal`
- Rate limit trigger must be disabled for seed scripts
- SVG planes must use viewport pixel coordinates (no viewBox) to render across screen sizes
- SVG animateMotion with rotate="auto" gives smooth tangent-following — CSS keyframes cause wobble
- Bundle is ~502 KB (chunk size warning) — consider code-splitting if it grows
